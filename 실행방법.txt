■ 시스템 개요
  - 콤프레샤 및 관련 설비의 에너지 낭비를 분석하고
    절감 효과를 예측하는 영업 시연용 데모 시스템입니다.

■ 폴더 구성
  KY_Energy_Demo_v5/
  ├── 1_EXE빌드하기.bat       ← EXE 파일 생성 (최초 1회만 실행)
  ├── 2_바로실행하기.bat       ← Python이 설치된 PC에서 즉시 실행
  ├── 실행방법.txt             ← 이 파일
  └── _src/                   ← 소스 파일 (건드리지 마세요)


══════════════════════════════════════════════════════════════════
 방법 1. 바로 실행 (가장 빠름) - Python만 있으면 됩니다
══════════════════════════════════════════════════════════════════

  ① Python이 설치되어 있는지 확인
    - Python이 없다면: https://www.python.org/downloads/
    - Python 3.10 이상 설치 권장

  ② 「2_바로실행하기.bat」파일을 더블클릭

  ③ 검은 창(명령 프롬프트)이 열리고 잠시 후
     브라우저에서 시스템이 자동으로 실행됩니다.

  ④ 브라우저 주소창에 자동으로 열리지 않으면
     직접 입력: http://localhost:5000

  ⑤ 시스템 종료 시: 검은 창(명령 프롬프트)을 닫으면 됩니다.


══════════════════════════════════════════════════════════════════
 방법 2. EXE 파일 생성 후 실행 (배포용, Python 불필요)
══════════════════════════════════════════════════════════════════

  ① 「1_EXE빌드하기.bat」파일을 더블클릭 (최초 1회)
    - 인터넷 연결 필요 (패키지 다운로드)
    - 빌드 시간: 약 3~10분 소요
    - 완료 후 dist\스마트에너지시스템\ 폴더가 생성됩니다.

  ② 빌드 완료 후:
    dist\스마트에너지시스템\스마트에너지시스템.exe 더블클릭

  ③ 이후에는 dist 폴더만 고객사에 전달하면 됩니다.
    (Python 설치 없이 실행 가능)

  ※ 주의: dist 폴더 내의 모든 파일이 함께 있어야 합니다.
          .exe 파일만 따로 복사하면 실행되지 않습니다.


══════════════════════════════════════════════════════════════════
 주요 기능
══════════════════════════════════════════════════════════════════

  1. 대시보드       - 전체 설비 현황 및 에너지 절감 KPI 요약
  2. 설비관리       - 설비 목록, 상세 정보, 등록/수정
  3. 운전데이터     - 일별 운전 현황 및 전력사용량 조회
  4. 에너지 분석    - 단위소비전력, 효율등급, 비효율 설비 분석
  5. 절감 시뮬레이션- 개선 전/후 비교, 투자 회수기간 분석
  6. 유지보수 관리  - 점검 이력, 정비 내역 관리
  7. 부품 재고관리  - 부품 현황, 입출고 이력


══════════════════════════════════════════════════════════════════
 최초 실행 안내
══════════════════════════════════════════════════════════════════

  - 데이터베이스(ky_energy.db)는 최초 실행 시 자동으로 생성됩니다.
  - 시연용 샘플 데이터가 자동으로 입력됩니다.
  - 별도로 설치하거나 설정할 것이 없습니다.
